## Agent Prompt 英文复刻方案

### 目标
将三个分析 Agent 的 Prompt 恢复为原始版本的英文版本，同时保留当前版本的所有优势功能（预计算、多时间框架、进度跟踪、性能监控）。

### 修改文件

1. **`indicator_agent.py`**
   - 恢复原始英文 HFT 风格 Prompt
   - 保持直接调用工具的方式
   - 保留预计算、多时间框架、进度跟踪

2. **`pattern_agent.py`**
   - 恢复原始英文 Prompt（16 种形态描述）
   - 保持直接调用工具的方式
   - 保留多时间框架、进度跟踪

3. **`trend_agent.py`**
   - 恢复原始英文 Prompt（支撑线/阻力线分析）
   - 保持直接调用工具的方式
   - 保留多时间框架、进度跟踪

4. **`decision_agent_original.py`**
   - 已存在，无需修改

### 保留的优势功能
- ✅ 直接工具调用（不让 LLM 决定）
- ✅ 预计算优化
- ✅ 多时间框架支持
- ✅ WebSocket 实时进度跟踪
- ✅ 性能监控装饰器
- ✅ 错误处理和重试机制
